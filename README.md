# 投資撥回試算與比較系統

這是一個基於 React 的單頁應用程式 (SPA)，專為財務分析設計，用於試算與比較投資型保單的撥回金額。系統符合 `speckit.specify` 規格需求，支援多情境比較與平均成本分析。

**狀態**: ✅ 已完成 (v1.0.0)

## 功能特色

- **核心計算**：精確計算實際投入單位數、撥回金額 (USD/TWD)。
- **多情境比較**：支援同時比較多個不同保費、匯率或費用率的設定。
- **平均成本分析**：自動計算投資組合的平均單位成本 (Average Unit Cost)。
- **響應式設計**：包含側邊欄控制面板與結果儀表板，適配行動裝置。
- **Demo 模式**：內建 6 筆標準驗證數據，一鍵載入。
- **匯出功能**：支援 CSV 匯出與列印優化。

## 快速開始

本專案採用 **Single File Architecture** (React + Tailwind via CDN)，無需安裝 Node.js 依賴即可直接執行。

### 啟動步驟

1. 下載或複製本專案目錄。
2. 直接使用瀏覽器 (Chrome, Edge, Firefox) 開啟 `index.html`。
3. 系統將自動載入並顯示主畫面。

### 測試

本專案包含核心邏輯的單元測試 `payout.test.js`。若需執行測試，請確保環境中有安裝 Node.js 與 Jest：

```bash
# 初始化並安裝 Jest
npm init -y
npm install jest

# 執行測試
npm test
```

或查看 `payout.test.js` 中的程式碼以了解測試邏輯。

## 系統邏輯說明

1. **實際投入單位數** = `目標保費 * (1 - 費用率) / (假設淨值 * 美元匯率)`
2. **撥回金額** = `實際單位數 * 撥回因子 * 美元匯率`
3. **分段撥回**：
   - 情境 A (淨值 > 上界): 使用因子 A (預設 0.042)
   - 情境 B (下界 <= 淨值 <= 上界): 使用因子 B (預設 0.036)
