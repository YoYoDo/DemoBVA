# Project Context

This project is about SEO Skills analysis.

# Your Role

When helping with this project, always:

1. Consider the context of Taiwan/Traditional Chinese market.
2. Focus on practical applications of SEO tools (GTM, GA4).

# Memory Protocol

When you learn something new that should be persisted:

1. Create a markdown file in `.agents/memory/`.
2. Name it descriptively, e.g., `project_naming_conventions.md`.
3. Add the new information there.
