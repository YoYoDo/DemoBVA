# 專案檢查清單：投資撥回試算與比較系統

## 核心邏輯與計算
- [x] **實際單位數計算**: `目標保費 * (1 - 費用率) / (淨值 * 進場匯率)`
- [ ] **撥回金額計算**:
    - [ ] 月撥回 (TWD) = `單位數 * 因子 * 現時匯率`
    - [ ] 年撥回 (TWD) = `月撥回 * 12`
- [x] **分段撥回因子**:
    - [ ] 情境 A: 淨值 >= 9 -> 因子 0.042
    - [ ] 情境 B: 淨值 < 9 -> 因子 0.036
- [x] **精度處理**:
    - [x] 內部運算使用浮點數。
    - [x] UI 顯示四捨五入至整數 (含千分位)。
    - [x] 提供切換顯示小數點後兩位的功能。
