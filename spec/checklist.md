# Checklist - 投資撥回試算與比較系統（新版）

## A. 核心計算
- [x] 實際單位數公式正確：`目標保費 * (1 - 費用率) / (NAV * 進場匯率)`
- [x] 情境 A（NAV >= 9）使用因子 `0.042`
- [x] 情境 B（8 <= NAV < 9）使用因子 `0.036`
- [x] 月撥回（TWD）公式正確：`實際單位數 * 因子 * 現時匯率`
- [x] 年撥回（TWD）公式正確：`月撥回 * 12`
- [x] 平均單位成本採累積加權計算
- [x] 內部計算不過早截斷
- [x] 顯示金額有千分位
- [x] 提供整數/小數兩位顯示切換

## B. 輸入與全域參數
- [x] 保費下拉含 30萬、50萬、100萬、200萬、300萬、500萬、1000萬
- [x] 保費支援自定義輸入
- [x] 費用率下拉含 3.0%、2.8%、2.7%
- [x] 每列有獨立進場匯率
- [x] Header 有全域現時匯率欄位
- [x] 修改現時匯率會即時更新所有列結果
