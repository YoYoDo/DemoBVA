# Specification: 投資撥回試算與比較系統

**Status**: Draft
**Feature**: Payout Calculator
**Short Name**: payout-calculator

## 1. 概述 (Overview)
開發一個名為「投資撥回試算與比較系統」的單頁應用程式 (SPA)。該系統旨在協助投資人試算與比較不同投入情境下的單位數取得與預估撥回金額。系統需支援多筆進場資料的彙整計算，並提供即時的匯率連動調整、資料自動儲存與 Excel 匯出功能，以利理財規劃與效益分析。

## 2. 使用者故事 (User Stories)
- **作為投資人**，我想要輸入目標保費、費用率與當時匯率，以便計算我實際取得的投資單位數。
- **作為投資人**，我想要根據不同的淨值 (NAV) 情境，預估每月的美元與台幣撥回金額，了解現金流狀況。
- **作為理財顧問**，我想要同時比較多筆不同時間點進場的投資組合，以便向客戶說明加權平均成本與總效益。
- **作為使用者**，我想要將試算結果匯出成 Excel 或列印，以便保存或進行紙本解說。
- **作為使用者**，我希望系統能自動儲存我的輸入資料，避免重新整理頁面後遺失。
- **作為使用者**，我想要能一鍵清除所有資料，以便重新開始新的試算。

## 3. 功能需求 (Functional Requirements)

### 3.1 核心計算邏輯
1.  **實際投入單位數**：
    $$實際單位數 = \frac{目標保費 \times (1 - 費用率)}{假設淨值 \times 進場匯率}$$
2.  **分段撥回規則** (系統預設)：
    -   **情境 A**：淨值 $\ge$ 9 $\rightarrow$ 撥回因子 **0.042**
    -   **情境 B**：8 $\le$ 淨值 < 9 $\rightarrow$ 撥回因子 **0.036**
3.  **撥回金額計算** (以台幣為最終產出)：
    -   月撥回 (TWD) = 實際單位數 $\times$ 撥回因子 $\times$ 現時匯率
    -   年撥回 (TWD) = 月撥回 (TWD) $\times$ 12
    -   **雙情境計算**：系統需同時計算並對比「情境 A ($\ge$ 9)」與「情境 B (< 9)」兩種狀況下的收益。
4.  **數值精度**：
    -   運算過程：保持浮點數精確度 (避免過早截斷)。
    -   顯示格式：金額需加上千分位分隔 (e.g., 1,000,000)。
    -   小數點處理：預設四捨五入至整數，提供切換開關顯示小數點後兩位。

### 3.2 參數設定與輸入
-   **目標保費 (Premium)**：
    -   預設選項：30萬、50萬、100萬、200萬、300萬、500萬、1,000萬。
    -   支援自定義數值輸入，預設為 100萬。
-   **費用率 (Fee Rate)**：
    -   下拉選單：3.0%、2.8% (預設)、2.7%。
-   **匯率設定**：
    -   **進場匯率 (Cost Exchange Rate)**：每筆資料獨立設定，用於計算單位成本。
    -   **現時匯率 (Current Exchange Rate)**：全域設定 (位於 Header)，變動時即時更新所有撥回 (TWD) 計算結果。預設值 31.5。
-   **淨值 (NAV)**：手動輸入，用於決定撥回因子區間與單位成本計算。

### 3.3 儀表板與列表 (Dashboard & List)
-   **組合總覽 (Portfolio Summary)**：
    -   顯示「加權平均單位成本」。
    -   顯示「總投入本金」。
    -   顯示「當前預估總撥回」與「預估年撥回」（需並列顯示情境 A 與情境 B 的金額）。
-   **動態表格** (Result Table)：
    -   每一列除了顯示進場基本資料外，需包含 **情境對照區**：
        -   **月撥回區塊**：分別列出「淨值 $\ge$ 9」與「8 $\le$ 淨值 < 9」時的台幣金額。
        -   **年撥回區塊**：分別列出「淨值 $\ge$ 9」與「8 $\le$ 淨值 < 9」時的台幣總額。
    -   支援即時編輯與刪除單筆資料。
    -   **清除功能**：提供「清除全部」按鈕，一鍵清空所有資料。
    -   **Demo 模式**：系統預設載入驗證數據以供展示。
-   **資料持久化**：
    -   使用 LocalStorage 自動儲存使用者輸入的投資組合資料。
    -   頁面重新載入時自動還原上次的資料。

### 3.4 輸出與互動
-   **匯出**：支援 Excel (.xlsx) 下載 (使用 SheetJS)。
-   **列印**：支援完整畫面列印。
-   **複製**：一鍵複製表格內容 (Clipboard API)。
-   **輔助說明**：每個欄位需有 Tooltip 說明計算邏輯與單位。
