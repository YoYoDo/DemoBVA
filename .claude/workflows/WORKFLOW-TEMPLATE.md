# 工作流模板 (Workflow Template)

## 使用說明

此檔案是所有工作流的標準模板。創建新工作流時：

1. 複製此目錄到新的工作流名稱
2. 修改以下部分：
   - 檔案開頭的 frontmatter（名稱、版本、描述等）
   - 所有標題和內容
   - 步驟編號和說明
3. 添加 `steps.json` 和 `checklist.md`
4. 在 `active-workflow.md` 中註冊

## 目標

此工作流的主要目標是什麼？應該達成什麼成果？

## 先決條件

在開始此工作流之前，需要：

- [ ] 環境已設置完成
- [ ] 必要的檔案已準備就緒
- [ ] 相關的技能已學習（參考 `.claude/memory/learned-skills.md`）
- [ ] 工作區已打開

## 提示詞模板

使用此提示詞執行此工作流：

```markdown
[執行工作流: workflow-name]

請執行此工作流：

1. 讀取 WORKFLOW.md 中的完整定義
2. 檢查 steps.json 中的步驟配置
3. 按順序執行每個步驟
4. 更新 checklist.md 進度
5. 完成時更新 active-workflow-state.md
6. 記錄到 workflow-history.md
```
