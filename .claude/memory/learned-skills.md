## 1. find-skills

- **Description**: Helps users discover and install agent skills when they ask questions like "how do I do X", "find a skill for X", "is there a skill that can...", or express interest in extending capabilities.
- **Path**: .claude/skills/find-skills
- **Use Cases**:
  - Finding functionality that might exist as an installable skill
  - Discovering domain-specific tools and workflows
  - Searching for agent capabilities
  - Installing and managing skills from the ecosystem
- **Key Commands**:
  - `npx skills find [query]` - Search for skills
  - `npx skills add <package>` - Install a skill
  - `npx skills check` - Check for updates
  - `npx skills update` - Update installed skills
